# Hot Homes

1 week project in an attempt to clone AirBnb.

### Description

_Front End_

* React, Redux, React-Router
* Bulma and Less

_Back End_

* Loopback for API endpoints
* MongoDB for database
* Express / Node.js for server

#### Set-up

* run npm start
